5.1 Learning objectives

By the end of this chapter, students should be able to:

  1. Distinguish between quantitative and categorical variables.
  2. Identify the four levels of measurement: nominal, ordinal, interval, and ratio.
  3. Explain the difference between independent and dependent variables.
  4. Describe the distinction between discrete and continuous variables.
  5. Explain why the type of variable determines the appropriate statistical methods.

5.2 Introduction

Every statistical analysis begins with data, and data consists of variables. A variable is any characteristic that can take different values across observations. Height, job title, monthly revenue, product rating, number of support tickets, region, all of these are variables.

Not all variables are alike. Whether a variable is numerical or categorical, whether it has a meaningful zero point, whether its values can be ordered, these properties determine which statistical methods apply. Using the wrong method for the variable type is one of the most common analytical errors in practice, and it produces results that are mathematically valid but substantively meaningless.

This chapter establishes the vocabulary and classifications that all subsequent chapters assume.

5.3 Quantitative and categorical variables

The first and most important distinction is between quantitative variables and categorical variables.

5.3.1 Quantitative variables

A quantitative variable takes numerical values that represent amounts or counts. Arithmetic operations, addition, subtraction, averaging, make sense on quantitative variables. The mean of a set of transaction values is a meaningful number. So is the difference between two delivery times, or the sum of units sold across regions.

Examples from management: - Revenue (in £) - Number of employees - Customer satisfaction score (if treated as numeric) - Delivery time in hours - Return on investment (%)

5.3.2 Categorical variables

A categorical variable places observations into groups or categories. The numbers used to code categories are labels, not quantities: if you code “Male” as 1 and “Female” as 2, the number 2 does not mean twice as much as 1. Arithmetic operations on category codes are meaningless.

Examples from management: - Department (Operations, Sales, Finance) - Product type - Region - Payment method - Whether a customer churned (Yes/No)

A special case worth noting: binary variables (also called dichotomous variables) have exactly two categories. Churned/not churned, passed/failed, promoted/not promoted. These often receive special treatment because they can be coded as 0 and 1, which enables specific methods such as logistic regression.

5.4 Levels of measurement

Within the two broad categories, variables are further classified by level of measurement, a framework developed by psychologist Stanley Stevens in 1946. The four levels are nominal, ordinal, interval, and ratio.

5.4.1 Nominal

Nominal variables name or label categories with no inherent order. Department names, product codes, countries, payment methods, none of these have a natural ranking. You can count how many observations fall into each category (frequency) and you can find the most common category (mode), but you cannot meaningfully order, subtract, or average nominal values.

5.4.2 Ordinal

Ordinal variables have categories with a meaningful order, but the gaps between categories are not necessarily equal. A five-point satisfaction rating from “Very dissatisfied” to “Very satisfied” is ordinal: you know that 4 is better than 3, but you cannot assume that the difference between 4 and 3 is the same size as the difference between 3 and 2.

This is the point most frequently violated in practice. Likert scale responses are ordinal. Reporting their mean and applying inferential tests designed for interval data is extremely common, and the debate about whether it is acceptable continues among statisticians. For this book: when in doubt, treat ordinal data as ordinal and use the appropriate methods (covered in Chapter 26).

5.4.3 Interval

Interval variables have a meaningful order and equal gaps between values, but no true zero point. Temperature in Celsius is the classic example: the difference between 10°C and 20°C is the same as the difference between 20°C and 30°C, but 0°C does not mean “no temperature.” You can meaningfully add and subtract, but ratios are not meaningful: 20°C is not twice as hot as 10°C.

Interval variables are less common in business data than they might appear. Calendar years and dates have interval properties (the gap between 2020 and 2022 equals the gap between 2022 and 2024) but no true zero.

5.4.4 Ratio

Ratio variables have a meaningful order, equal intervals, and a true zero, meaning that zero represents a genuine absence of the quantity. Revenue, weight, age, distance, and time elapsed all have this property. When revenue is zero, there is no revenue. Ratios are meaningful: a company with £10 million in revenue has twice the revenue of one with £5 million.

Most quantitative business variables are ratio variables. This is the highest level of measurement and supports the full range of arithmetic operations.

5.4.5 Summary of levels

Level Order Equal intervals True zero Example
Nominal No No No Department name
Ordinal Yes No No Customer rating (1–5)
Interval Yes Yes No Temperature (°C)
Ratio Yes Yes Yes Revenue (£)

5.5 Discrete and continuous variables

Quantitative variables are further divided into discrete and continuous.

A discrete variable takes a countable number of values, usually whole numbers. The number of complaints received, the number of products on a shelf, the number of employees absent on a given day, these can only be integers. You cannot have 3.7 complaints.

A continuous variable can theoretically take any value within a range, including fractions. Time, weight, temperature, and distance are continuous: a delivery can take 2.37 hours, a product can weigh 0.842 kg.

In practice, the distinction matters for probability calculations. The probability that a continuous variable takes exactly one specific value is zero, because there are infinitely many possible values. Probabilities for continuous variables are always defined over ranges, not points.

5.6 Independent and dependent variables

When a study examines a relationship between variables, it typically distinguishes between the independent variable and the dependent variable.

The dependent variable (DV) is the outcome being measured, what you are trying to explain or predict. In most management research, the DV is the thing the organisation cares about: sales, customer satisfaction, employee retention, defect rate.

The independent variable (IV) is the variable expected to influence or explain the dependent variable, what you think might cause or predict changes in the DV. In an experiment, the independent variable is what is deliberately manipulated. In observational research, it is what is being tested as a potential predictor.

Some examples:

Study question Independent variable Dependent variable
Does training increase sales performance? Whether the employee received training Sales performance score
Does store layout affect transaction value? Layout type (old/new) Average transaction value (£)
Does delivery speed predict repeat purchases? Delivery time (hours) Whether customer made a repeat purchase

In regression analysis, the independent variable is often called the predictor or explanatory variable, and the dependent variable is called the response or outcome variable.

The labels “independent” and “dependent” are not always an accurate description of causation, in observational data, you cannot always be sure which direction causation runs. But the labels identify the analytical role each variable plays: the IV is what you put on the right side of the equation; the DV is what you are trying to account for.

5.7 Why variable type matters for method selection

The level of measurement and variable type determine which statistical methods are appropriate. A brief preview:

  • Nominal DV: frequency analysis, chi-square tests, logistic regression.
  • Ordinal DV: median, percentiles, non-parametric tests (Mann-Whitney, Kruskal-Wallis).
  • Interval or ratio DV: mean, standard deviation, t-tests, ANOVA, linear regression.

Computing the mean of an ordinal variable may not be mathematically impossible, but it is analytically dubious. Applying a t-test to a nominal outcome is simply incorrect. The method must match the data.

5.8 Worked example

Code 
# A dataset from a hotel chain, illustrating variable types in a real context
set.seed(303)

hotel_data <- data.frame(
  guest_id      = 1:200,
  # Nominal
  room_type     = sample(c("Standard", "Deluxe", "Suite"), 200, replace = TRUE,
                          prob = c(0.55, 0.30, 0.15)),
  payment       = sample(c("Card", "Cash", "Points"), 200, replace = TRUE,
                          prob = c(0.70, 0.10, 0.20)),
  # Ordinal
  satisfaction  = sample(1:5, 200, replace = TRUE,
                          prob = c(0.05, 0.10, 0.20, 0.35, 0.30)),
  # Ratio
  spend_gbp     = round(rnorm(200, mean = 180, sd = 55), 2),
  nights        = sample(1:7, 200, replace = TRUE,
                          prob = c(0.30, 0.25, 0.20, 0.10, 0.07, 0.05, 0.03)),
  # Binary (special case of nominal)
  returned      = sample(c(TRUE, FALSE), 200, replace = TRUE, prob = c(0.45, 0.55))
)

# Appropriate summaries by variable type
cat("--- Nominal: room_type ---\n")
--- Nominal: room_type ---
Code 
print(table(hotel_data$room_type))

  Deluxe Standard    Suite 
      76       99       25
Code 
cat("\n--- Ordinal: satisfaction ---\n")

--- Ordinal: satisfaction ---
Code

Code 
cat("Median satisfaction:", median(hotel_data$satisfaction), "\n")
Median satisfaction: 4
Code

--- Ratio: spend_gbp ---
Code

5.9 Summary

A variable is any characteristic that varies across observations. Variables are classified as quantitative (numerical amounts) or categorical (group labels). Quantitative variables divide further into discrete (countable values) and continuous (any value within a range).

The level of measurement, nominal, ordinal, interval, or ratio, determines what operations make sense. Nominal variables name categories with no order. Ordinal variables have order but unequal gaps. Interval variables have equal gaps but no true zero. Ratio variables have all of the above plus a meaningful zero.

In any study examining relationships, the dependent variable is the outcome being explained, and the independent variable is the potential predictor or cause. Variable type governs which statistical methods are appropriate: using the wrong method for the variable type produces results that are technically computed but substantively wrong.
