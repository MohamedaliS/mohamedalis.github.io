15.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Construct and interpret a scatter plot for two quantitative variables.
  2. Describe the direction, form, and strength of a relationship from a scatter plot.
  3. Calculate and interpret Pearson’s correlation coefficient.
  4. Explain the limitations of correlation, including sensitivity to outliers.
  5. Distinguish correlation from causation.

15.2 Introduction

Many of the most important questions in management concern relationships between variables. Does higher advertising spending produce higher sales? Do employees who receive more training achieve higher productivity? Does customer satisfaction predict repeat purchases?

Scatter plots and correlation provide the first tools for investigating these questions. They reveal whether a relationship exists, what direction it takes, and how consistently the two variables move together. This chapter works with performance data from 120 retail stores, with variables covering advertising spend, footfall, weekly sales, promotional activity, and competitive environment.

15.3 Scatter plots

A scatter plot displays the joint distribution of two quantitative variables. Each observation is plotted as a point, with one variable on the horizontal axis and the other on the vertical axis. The resulting cloud of points reveals the pattern of the relationship.

Code 
library(dplyr)
ggplot(df, aes(x = ad_spend_gbp, y = weekly_sales_gbp)) +
  geom_point(colour = "#1B3A5C", alpha = 0.65, size = 2.2) +
  geom_smooth(method = "lm", se = TRUE, colour = "#C0392B",
              linewidth = 0.9) +
  scale_x_continuous(labels = scales::comma) +
  scale_y_continuous(labels = scales::comma) +
  labs(x = "Weekly advertising spend (GBP)",
       y = "Weekly sales (GBP)",
       title = "Advertising spend and weekly sales across 120 stores") +
  theme_dmm()

The shaded band around the trend line is the 95% confidence region for the line’s position. Points more spread around the line indicate more noise in the relationship.

15.3.1 What to look for in a scatter plot

Three features should be assessed when reading a scatter plot.

Direction. A positive relationship means that as one variable increases, the other tends to increase. A negative relationship means that as one increases, the other tends to decrease.

Form. Most business relationships are approximately linear — the points cluster around a straight line. Non-linear forms are possible but require different analytical tools.

Strength. A strong relationship means the points cluster tightly around the trend. A weak relationship means they are widely scattered with only a vague pattern.

15.4 Pearson’s correlation coefficient

The scatter plot describes a relationship visually. Pearson’s correlation coefficient, denoted r, quantifies its strength and direction in a single number.

\[r = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum_{i=1}^{n}(x_i - \bar{x})^2 \sum_{i=1}^{n}(y_i - \bar{y})^2}}\]

The coefficient r always lies between -1 and 1. A value of 1 indicates a perfect positive linear relationship; -1 a perfect negative one; 0 no linear relationship.

Code 
r_ads_sales    <- cor(df$ad_spend_gbp, df$weekly_sales_gbp)
r_foot_sales   <- cor(df$weekly_footfall, df$weekly_sales_gbp)
r_ads_foot     <- cor(df$ad_spend_gbp, df$weekly_footfall)

cor_table <- data.frame(
  Pair = c("Ad spend vs Weekly sales",
           "Footfall vs Weekly sales",
           "Ad spend vs Footfall"),
  r    = round(c(r_ads_sales, r_foot_sales, r_ads_foot), 3)
)

knitr::kable(cor_table,
  col.names = c("Variable pair", "Pearson r"),
  caption   = "Table 15.1: Correlation coefficients for key variable pairs") |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 15.1: Correlation coefficients for key variable pairs
Variable pair Pearson r
Ad spend vs Weekly sales 0.661
Footfall vs Weekly sales 0.831
Ad spend vs Footfall 0.642
Table 15.2: Conventional interpretation of correlation strength
Range of &#124;r&#124; Conventional label
&#124;r&#124; >= 0.90 Very strong
0.70 <= &#124;r&#124; < 0.90 Strong
0.50 <= &#124;r&#124; < 0.70 Moderate
0.30 <= &#124;r&#124; < 0.50 Weak
&#124;r&#124; < 0.30 Very weak or none

These labels are conventions, not thresholds. What counts as a strong correlation depends on the field and the question being asked. In social sciences, r = 0.40 may be noteworthy. In precision engineering, r = 0.95 might be considered modest.

15.5 A correlation matrix

When several variables are available, a correlation matrix provides an overview of all pairwise relationships.

Code 
numeric_vars <- df |>
  select(ad_spend_gbp, weekly_footfall, weekly_sales_gbp, months_open)

cor_matrix <- round(cor(numeric_vars), 3)

knitr::kable(cor_matrix,
  caption = "Table 15.3: Correlation matrix for numeric store variables") |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 15.3: Correlation matrix for numeric store variables
ad_spend_gbp weekly_footfall weekly_sales_gbp months_open
ad_spend_gbp 1.000 0.642 0.661 0.149
weekly_footfall 0.642 1.000 0.831 0.201
weekly_sales_gbp 0.661 0.831 1.000 0.232
months_open 0.149 0.201 0.232 1.000

The diagonal is always 1 (each variable is perfectly correlated with itself). The matrix is symmetric: the value above the diagonal equals the value below.

15.6 The effect of subgroups and context

Correlation values can differ substantially across subgroups.

Code 
cor_by_format <- df |>
  group_by(store_format) |>
  summarise(
    n = n(),
    r_ads_sales = round(cor(ad_spend_gbp, weekly_sales_gbp), 3)
  )

knitr::kable(cor_by_format,
  col.names = c("Store format", "n",
                "r (ad spend vs sales)"),
  caption   = "Table 15.4: Correlation by store format") |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 15.4: Correlation by store format
Store format n r (ad spend vs sales)
High Street 46 0.662
Online 10 0.839
Retail Park 27 0.719
Superstore 37 0.608

A correlation calculated across all stores may hide different patterns within store types. Pooling data from groups with different baseline characteristics can inflate or deflate the overall coefficient.

15.7 Limitations of correlation

15.7.1 Sensitivity to outliers

A single extreme observation can substantially change the value of r.

Code 
# Correlation with and without the top-spending store
r_full    <- cor(df$ad_spend_gbp, df$weekly_sales_gbp)
df_trim   <- df |> filter(ad_spend_gbp < quantile(ad_spend_gbp, 0.98))
r_trimmed <- cor(df_trim$ad_spend_gbp, df_trim$weekly_sales_gbp)

outlier_table <- data.frame(
  Dataset = c("All 120 stores", "Top 2% ad spenders removed"),
  r       = round(c(r_full, r_trimmed), 3),
  n       = c(nrow(df), nrow(df_trim))
)
knitr::kable(outlier_table,
  col.names = c("Dataset", "r", "n"),
  caption   = "Table 15.5: Effect of outliers on correlation") |>
  kableExtra::kable_styling(bootstrap_options = c("striped"),
                             full_width = FALSE)
Table 15.5: Effect of outliers on correlation
Dataset r n
All 120 stores 0.661 120
Top 2% ad spenders removed 0.641 117

15.7.2 Correlation measures only linear relationships

Pearson’s r measures the strength of the linear component of a relationship. If two variables have a strong curved (non-linear) relationship, r may be near zero even though the variables are clearly related. Always inspect the scatter plot before relying on r.

Colouring by a third variable reveals that promotion-active stores have a different intercept in the scatter plot, meaning they achieve higher sales at the same level of advertising spend. A single correlation coefficient would fail to capture this nuance.

15.8 Worked example

Note

Scenario. A regional director wants to know whether stores with more months of operation are associated with higher weekly sales. Calculate the relevant correlation coefficient, plot the relationship, and interpret the result.

Code 
r_months_sales <- cor(df$months_open, df$weekly_sales_gbp)
cat("r (months open vs weekly sales) =", round(r_months_sales, 3), "\n")
r (months open vs weekly sales) = 0.232
Code 
ggplot(df, aes(x = months_open, y = weekly_sales_gbp)) +
  geom_point(colour = "#1B3A5C", alpha = 0.65, size = 2.0) +
  geom_smooth(method = "lm", se = FALSE,
              colour = "#C0392B", linewidth = 0.9) +
  scale_y_continuous(labels = scales::comma) +
  labs(x = "Months open", y = "Weekly sales (GBP)",
       title = "Store maturity and weekly sales") +
  theme_dmm()

The correlation coefficient and scatter plot together show both the direction and strength of the relationship. A positive but modest r suggests that time open is one factor among several influencing sales performance. The scatter plot reveals whether the relationship is approximately linear and whether any stores stand out as unusual.

15.9 Summary

A scatter plot displays the relationship between two quantitative variables. Key features to describe are direction (positive or negative), form (linear or non-linear), and strength (tight or loose clustering). Pearson’s correlation coefficient r quantifies linear strength and direction on a scale from -1 to 1.

Correlation has important limitations. It is sensitive to outliers, measures only linear relationships, can differ across subgroups, and does not establish causation. A scatter plot should always accompany or precede any correlation calculation to avoid misinterpretation.

15.10 Key terms

Table 15.6: Key terms for Chapter 15
Term Definition

Positive correlation A relationship in which higher values of one variable tend to coincide with higher values of the other.
Negative correlation A relationship in which higher values of one variable tend to coincide with lower values of the other.
Pearson's r A coefficient between -1 and 1 measuring the strength and direction of a linear relationship.
Correlation matrix A table showing Pearson's r for all pairs of numeric variables in a dataset.

  2. A scatter plot of advertising spend against weekly sales shows a positive trend but several points are far from the line. A colleague reports r = 0.63 and says “advertising spend explains 63% of the variation in sales.” What is wrong with this interpretation?

  3. Using Table 15.4, the correlation between ad spend and sales varies by store format. What does this imply for how the company should analyse and act on this relationship?

  5. A dataset shows r = 0.05 between two variables. A manager concludes: “There is no relationship between these variables.” Under what circumstances would this conclusion be wrong?

## Limitations of correlation

### Sensitivity to outliers

A single extreme observation can substantially change the value of r.
