25.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Distinguish the chi-square goodness-of-fit test from the chi-square test of independence.
  2. Construct a contingency table and compute observed and expected frequencies.
  3. Conduct and interpret a chi-square test of independence in R.
  4. Interpret Cramér’s V as an effect size for categorical associations.
  5. State the assumptions and limitations of chi-square tests.

25.2 Introduction

All tests in Chapters 22 to 24 compared means of numerical variables. Many business questions concern categorical variables: is purchasing channel associated with customer segment? Do customer complaints differ by product category? Do employee turnover rates vary by department?

The chi-square test evaluates whether an observed distribution of counts differs from an expected distribution, or whether two categorical variables are associated. The dataset for this unit contains 400 customer records with variables covering segment (Budget, Mid-range, Premium) and preferred channel (Online, In-store, Both).

25.3 The chi-square test of independence

The chi-square test of independence evaluates whether two categorical variables are associated in the population. The null hypothesis is that the variables are independent.

\[H_0: \text{the two variables are independent}\] \[H_1: \text{the two variables are associated}\]

25.5 The test statistic

The chi-square statistic compares observed frequencies with the frequencies expected under independence:

\[\chi^2 = \sum_{\text{all cells}} \frac{(O - E)^2}{E}\]

where \(O\) is the observed count and \(E\) is the expected count if \(H_0\) were true:

\[E = \frac{\text{row total} \times \text{column total}}{\text{grand total}}\]

Code 
chi_result <- chisq.test(cont_table, correct = FALSE)

chi_output <- data.frame(
  Element = c("Chi-square statistic", "Degrees of freedom",
              "p-value", "Decision (alpha = 0.05)"),
  Value   = c(round(chi_result$statistic, 3),
              chi_result$parameter,
              round(chi_result$p.value, 4),
              ifelse(chi_result$p.value <= 0.05,
                     "Reject H0: variables are associated",
                     "Fail to reject H0"))
)

knitr::kable(chi_output,
  col.names = c("Element","Value"),
  caption   = "Table 25.3: Chi-square test of independence") |>
  kableExtra::kable_styling(bootstrap_options = c("striped"),
                             full_width = FALSE)

25.7 Effect size: Cramér’s V

The chi-square statistic grows with sample size, so it cannot be used directly as an effect size. Cramér’s V standardises it:

\[V = \sqrt{\frac{\chi^2}{n \cdot \min(r-1, c-1)}}\]

where \(r\) and \(c\) are the numbers of rows and columns.

Code 
n    <- sum(cont_table)
k    <- min(nrow(cont_table) - 1, ncol(cont_table) - 1)
V    <- sqrt(chi_result$statistic / (n * k))
cat("Cramér's V =", round(V, 3), "\n")
Cramér's V = 0.183
Code 
cat("Interpretation:",
    ifelse(V < 0.1, "Negligible",
           ifelse(V < 0.3, "Small",
                  ifelse(V < 0.5, "Moderate","Strong"))), "\n")
Interpretation: Small

25.10 Assumptions

Table 25.5: Chi-square test assumptions
Assumption Detail
Independent observations Each observation belongs to exactly one cell; no repeated measures.
Expected frequency >= 5 per cell If any expected cell count is below 5, use Fisher's exact test or collapse categories.
Fixed margins The chi-square test treats both row and column totals as fixed by the study design.

25.11 Summary

The chi-square test of independence evaluates whether two categorical variables are associated. The test statistic compares observed cell frequencies to those expected under independence. Degrees of freedom equal \((r-1)(c-1)\). Cramér’s V provides a sample-size-independent effect size. The test requires that expected cell counts are at least 5 in all cells.

25.12 Key terms

Table 25.6: Key terms for Chapter 25
Term Definition
Chi-square test of independence A test evaluating whether two categorical variables are associated.
Contingency table A cross-tabulation showing the joint frequency distribution of two categorical variables.
Expected frequency The cell count predicted under H0 (independence): row total x column total / grand total.
Cramér's V A standardised effect size for chi-square: sqrt(chi-squared / (n x min(r-1, c-1))).
Goodness-of-fit test A chi-square test comparing observed frequencies to a specified theoretical distribution.

## Effect size: Cramér's V

The chi-square statistic grows with sample size, so it cannot be used directly as an effect size. Cramér's V standardises it:

$$V = \sqrt{\frac{\chi^2}{n \cdot \min(r-1, c-1)}}$$

where $r$ and $c$ are the numbers of rows and columns.

```{r}
#| label: ch25-cramers-v
#| echo: true
n    <- sum(cont_table)
k    <- min(nrow(cont_table) - 1, ncol(cont_table) - 1)
V    <- sqrt(chi_result$statistic / (n * k))
cat("Cramér's V =", round(V, 3), "\n")
cat("Interpretation:",
    ifelse(V < 0.1, "Negligible",
           ifelse(V < 0.3, "Small",
                  ifelse(V < 0.5, "Moderate","Strong"))), "\n")
```
