19.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Construct a confidence interval for a population proportion.
  2. Interpret a proportion confidence interval correctly.
  3. Determine the sample size required for a desired margin of error.
  4. Compare confidence intervals for proportions across subgroups.
  5. Explain the role of the conservative estimate \(\hat{p} = 0.5\) in sample size planning.

19.2 Introduction

Many business questions concern proportions rather than means. What fraction of customers are satisfied? What proportion of products pass quality inspection? What percentage of survey respondents would recommend the company to others?

Confidence intervals for proportions follow the same logic as those for means but use the normal approximation rather than the t-distribution. Sample size planning, which determines how large a survey needs to be to achieve a desired precision, is one of the most practical applications of this chapter.

The dataset for this unit contains 500 customer satisfaction survey responses, with variables covering channel, age group, satisfaction status, NPS score, and complaint history.

19.3 Confidence interval for a proportion

Let \(\hat{p}\) be the sample proportion (the fraction of respondents with a given characteristic). The confidence interval for the population proportion \(p\) is:

\[\hat{p} \pm z^* \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}\]

where \(z^*\) is the critical value from the standard normal distribution (1.96 for 95% confidence) and the term under the square root is the standard error of \(\hat{p}\).

This formula relies on the normal approximation to the binomial, which is reliable when \(n\hat{p} \geq 10\) and \(n(1-\hat{p}) \geq 10\).

Code 
library(dplyr)

n_total   <- nrow(df)
n_satisfied <- sum(df$satisfied == "Yes")
p_hat     <- n_satisfied / n_total
se_p      <- sqrt(p_hat * (1 - p_hat) / n_total)
z_star    <- qnorm(0.975)   # 95% CI

lower_95 <- p_hat - z_star * se_p
upper_95 <- p_hat + z_star * se_p

prop_ci <- data.frame(
  Statistic = c("n", "Satisfied (count)", "p-hat",
                "Standard error", "z* (95%)",
                "Lower 95% CI", "Upper 95% CI"),
  Value = round(c(n_total, n_satisfied, p_hat,
                  se_p, z_star, lower_95, upper_95), 4)
)

knitr::kable(prop_ci,
  col.names = c("Statistic", "Value"),
  caption   = "Table 19.1: 95% CI for customer satisfaction proportion") |>
  kableExtra::kable_styling(bootstrap_options = c("striped"),
                             full_width = FALSE)

n 500.0000

Interpretation. Based on 500 survey responses, we estimate that between 72.9% and 80.3% of the customer population is satisfied. The procedure used produces intervals containing the true proportion 95% of the time.

19.6 Sample size planning for proportions

Before collecting data, a researcher often needs to determine how large a sample is required to achieve a desired margin of error \(E\) at a given confidence level.

\[n = \left(\frac{z^*}{E}\right)^2 \hat{p}(1-\hat{p})\]

When no prior estimate of \(p\) is available, the conservative choice \(\hat{p} = 0.5\) maximises the required sample size. This is conservative because \(p(1-p)\) is maximised at \(p = 0.5\).

Code 
# Required sample size for various margins of error at 95% confidence
z_star <- 1.96
p_est  <- 0.5   # conservative

ss_table <- do.call(rbind, lapply(c(0.01, 0.02, 0.03, 0.05, 0.10), function(E) {
  n_required <- ceiling((z_star / E)^2 * p_est * (1 - p_est))
  data.frame(
    Margin_of_error = paste0(E * 100, "%"),
    n_required      = n_required
  )
}))

knitr::kable(ss_table,
  col.names = c("Desired margin of error", "Required sample size (n)"),
  caption   = "Table 19.4: Sample size requirements (95% CI, p = 0.5)") |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 19.4: Sample size requirements (95% CI, p = 0.5)
Desired margin of error Required sample size (n)
1% 9604
2% 2401
3% 1068
5% 385
10% 97

A margin of error of ±3 percentage points at 95% confidence requires approximately 1,067 respondents. This is the basis for the “around 1,000 respondents” standard used in many national polls.

19.6.1 Using prior information to reduce required sample size

Code 
# If prior data suggest p is close to 0.75, the required n is smaller
E      <- 0.03
p_prior <- 0.75

n_conservative <- ceiling((z_star / E)^2 * 0.5 * 0.5)
n_informed     <- ceiling((z_star / E)^2 * p_prior * (1 - p_prior))

prior_table <- data.frame(
  Approach = c("Conservative (p = 0.5)",
               paste0("Informed (p = ", p_prior, ")")),
  n_required = c(n_conservative, n_informed),
  Saving     = c(0, n_conservative - n_informed)
)

knitr::kable(prior_table,
  col.names = c("Approach", "Required n", "Savings vs conservative"),
  caption   = "Table 19.5: Sample size reduction with prior information") |>
  kableExtra::kable_styling(bootstrap_options = c("striped"),
                             full_width = FALSE)

Conservative (p = 0.5) 1068 0
Informed (p = 0.75) 801 267

19.7 Summary

A confidence interval for a proportion is \(\hat{p} \pm z^* \sqrt{\hat{p}(1-\hat{p})/n}\), where \(z^*\) is the standard normal critical value. The interval estimates the population proportion and communicates the uncertainty from sampling.

Sample size planning uses the same formula in reverse: given a desired margin of error and confidence level, calculate the minimum n required. The conservative estimate \(\hat{p} = 0.5\) maximises the required sample size and is used when no prior information is available.

19.8 Key terms

Table 19.6: Key terms for Chapter 19
Term Definition
Proportion CI A confidence interval for a population proportion: p-hat +/- z* sqrt(p-hat(1-p-hat)/n).
Standard error of p-hat The estimated standard deviation of the sampling distribution of p-hat: sqrt(p-hat(1-p-hat)/n).
Margin of error Half the width of a confidence interval; equal to z* times the standard error.
Sample size formula The formula n = (z*/E)^2 * p-hat*(1-p-hat), giving the minimum sample size for a desired margin of error E.
Conservative estimate Setting p-hat = 0.5 in the sample size formula to maximise n and ensure the required precision regardless of the true proportion.

## Sample size planning for proportions

Before collecting data, a researcher often needs to determine how large a sample is required to achieve a desired margin of error $E$ at a given confidence level.

$$n = \left(\frac{z^*}{E}\right)^2 \hat{p}(1-\hat{p})$$

```{r}

ss_table <- do.call(rbind, lapply(c(0.01, 0.02, 0.03, 0.05, 0.10), function(E) {

  data.frame(
    Margin_of_error = paste0(E * 100, "%"),
    n_required      = n_required

knitr::kable(ss_table,
  col.names = c("Desired margin of error", "Required sample size (n)"),
  caption   = "Table 19.4: Sample size requirements (95% CI, p = 0.5)") |>
```
