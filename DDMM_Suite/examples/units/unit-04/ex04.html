Worked Example 04: Choosing and Evaluating Sampling Methods

Context

The population for this unit is a workforce of 600 employees drawn from three departments: Operations (300 staff), Sales (200 staff), and Finance (100 staff). We will simulate four sampling approaches, compare the samples they produce, and evaluate which most faithfully represents the population on key variables.

Step 1: The population structure

library(dplyr)
pop_summary <- df |>
  group_by(department) |>
  summarise(
    n                = n(),
    pct              = round(100 * n() / nrow(df), 1),
    mean_tenure      = round(mean(tenure_years), 1),
    mean_engagement  = round(mean(engagement), 2)
  )

Step 2: Simulate four sampling methods (n = 60)

set.seed(99)
n_total <- 60

# Simple random sample
srs <- df[sample(nrow(df), n_total), ]

# Systematic sample (every 10th record)
sys <- df[seq(1, nrow(df), by = 10), ]

# Stratified sample (proportional allocation)
ops_n  <- round(n_total * 300/600)
sal_n  <- round(n_total * 200/600)
fin_n  <- n_total - ops_n - sal_n
strat  <- bind_rows(
  df[df$department == "Operations", ][sample(300, ops_n), ],
  df[df$department == "Sales",      ][sample(200, sal_n), ],
  df[df$department == "Finance",    ][sample(100, fin_n), ]
)

# Convenience sample (first 60 records)
conv <- df[1:n_total, ]

label_summary <- function(samp, label) {
  samp |>
    summarise(
      Method           = label,
      n                = n(),
      pct_operations   = round(100 * mean(department == "Operations"), 1),
      mean_tenure      = round(mean(tenure_years), 1),
      mean_engagement  = round(mean(engagement), 2)
    )
}

comparison <- bind_rows(
  label_summary(df, "Population (N=600)"),
  label_summary(srs, "Simple random (n=60)"),
  label_summary(sys, "Systematic (n=60)"),
  label_summary(strat, "Stratified (n=60)"),
  label_summary(conv, "Convenience (n=60)")
)

Step 4: Evaluate the methods

Table E4.3: Evaluation of sampling methods
Method Strengths Weaknesses
Simple random Unbiased; every member has equal probability of selection. Requires a complete sampling frame; can be logistically difficult for large populations.
Systematic Easy to implement; approximately random if the list is not ordered by the variable of interest. Biased if the list has a cyclical structure that coincides with the sampling interval.
Stratified Ensures proportional representation of known subgroups; often more precise than SRS. Requires prior knowledge of strata; more complex to administer.
Convenience Quick and cheap; no sampling frame required. Almost certainly biased; results cannot be generalised to the population.

Reflection questions

  1. In Table E4.2, which method most closely matches the population on all three variables? Does this surprise you?
  2. The convenience sample uses the first 60 records. Why might this be a particularly poor strategy with this dataset?
  3. If you could only ask 30 questions in a staff survey, which sampling method would you recommend? Justify your answer.
