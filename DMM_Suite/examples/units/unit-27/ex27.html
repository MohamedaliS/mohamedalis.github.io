Worked Example 27: End-to-End Analysis - Selecting, Running, and Reporting Five Tests

Context

You are an analyst preparing a workforce intelligence report for an HR director. The dataset contains 300 employee records from a single organisation. You must answer five questions, selecting the appropriate test for each, running it in R, and producing a complete report.

Step 2: Apply the test selection framework

Table E27.1: Test selection for each question
Question Outcome Groups Test
Is mean salary above the GBP 30,000 industry benchmark? Continuous 1 vs target One-sample t-test
Do trained employees have higher productivity? Continuous 2 indep Independent t-test (Welch)
Does productivity differ across departments? Continuous 5 indep One-way ANOVA + Tukey
Do employees who intend to leave have lower engagement? Ordinal 2 indep Mann-Whitney U
Is training completion associated with satisfaction band? Categorical 2 x 4 table Chi-square independence

Step 3: Run all five tests

# Q1: Salary vs benchmark
t1  <- t.test(df$salary_gbp, mu = 30000)
d1  <- (mean(df$salary_gbp) - 30000) / sd(df$salary_gbp)

# Q2: Training effect on productivity
yes <- df$productivity[df$training_completed == "Yes"]
no  <- df$productivity[df$training_completed == "No"]
t2  <- t.test(yes, no, var.equal = FALSE)
sp  <- sqrt(((length(yes)-1)*var(yes) + (length(no)-1)*var(no)) /
              (length(yes)+length(no)-2))
d2  <- (mean(yes) - mean(no)) / sp

# Q3: Productivity by department
anova_m <- aov(productivity ~ department, data = df)
a_tbl   <- summary(anova_m)[[1]]
eta2    <- a_tbl$`Sum Sq`[1] / sum(a_tbl$`Sum Sq`)

# Q4: Engagement by turnover intention
mw <- wilcox.test(engagement_score ~ turnover_intention,
                   data = df, exact = FALSE)

# Q5: Training vs satisfaction band
ct  <- table(df$training_completed, df$satisfaction_band)
chi <- chisq.test(ct, correct = FALSE)
V   <- sqrt(chi$statistic / (nrow(df) * min(nrow(ct)-1, ncol(ct)-1)))

Reflection questions

  1. For each analysis in Table E27.1, verify that the test selection is correct using the four-question framework.
