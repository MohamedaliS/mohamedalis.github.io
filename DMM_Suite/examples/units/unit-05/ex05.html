Worked Example 05: Identifying and Handling Variable Types

Context

This dataset contains records for 300 hotel guests across several properties. It includes a range of variable types: nominal, ordinal, interval, ratio, discrete, and continuous. The goal of this example is to correctly classify each variable, choose the appropriate summary statistic, and produce a suitable visual for each.

Step 4: Common mistakes to avoid

Table E5.5: Variable type mistakes and their consequences
Common mistake Why it matters
Calculating the mean of an ordinal variable (e.g. average satisfaction). Ordinal intervals are not equal. The mean of 'Low, Medium, High' codes is arithmetically possible but not meaningful.
Drawing a histogram for a nominal variable. A histogram implies continuous numeric data. Nominal categories have no natural order or spacing.
Treating a binary variable (0/1) as if its mean is a standard average. The mean of a binary variable is a proportion, which is valid, but must be interpreted as such.
Applying parametric tests to nominal data. Parametric tests assume at least interval-level data with a particular distribution.
