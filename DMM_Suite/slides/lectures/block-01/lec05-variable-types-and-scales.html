Variable Types, Independent and Dependent Variables, and Measurement Scales

Unit 05 | Data-Driven Management

Why variable type matters

The type of variable determines which statistical methods are valid.

Using the wrong method → results that are mathematically computed but substantively meaningless.

Two main categories

Quantitative Categorical
Numerical amounts or counts Group labels
Arithmetic is meaningful Arithmetic on codes is not meaningful
Revenue, hours, units sold Department, region, payment method

Four levels of measurement

Level Order Equal gaps True zero Example
Nominal No No No Department
Ordinal Yes No No Satisfaction (1–5)
Interval Yes Yes No Temperature (°C)
Ratio Yes Yes Yes Revenue (£)

Nominal

  • Categories with no inherent order
  • “Department A” is not less than “Department B”
  • Valid operations: count (frequency), mode
  • Invalid: mean, median, subtraction

Ordinal

  • Categories have a meaningful order
  • Gaps between categories are not assumed equal
  • “4 out of 5” is better than “3 out of 5”, but by how much?

Warning

Likert scale means are widely reported but assume equal intervals. Handle with care.

Interval and ratio

Interval: equal gaps, no true zero
- Can add and subtract; ratios are not meaningful
- 20°C is not “twice as hot” as 10°C

Ratio: equal gaps, true zero present
- Full arithmetic applies
- £20m revenue is twice £10m revenue
- Most quantitative business variables are ratio

Discrete vs continuous

Discrete: countable values, usually whole numbers
Number of complaints, number of staff, units sold

Continuous: any value within a range
Delivery time, weight, temperature

This matters for probability calculations (Chapter 13 onwards).

Independent and dependent variables

Role Symbol
Dependent variable (DV) Outcome being explained Y
Independent variable (IV) Potential predictor or cause X

Examples

Study question IV DV
Does training improve sales? Training (yes/no) Sales performance score
Does layout affect spend? Layout type Average transaction (£)
Does delivery speed predict repeat purchase? Delivery time (hrs) Repeat purchase (yes/no)

Method selection by variable type

DV type Appropriate methods
Nominal Frequency, chi-square, logistic regression
Ordinal Median, non-parametric tests (Chapters 20–26)
Interval / Ratio Mean, SD, t-tests, ANOVA, regression

In R, variable classification matters

set.seed(303)
df <- data.frame(
  room_type    = factor(sample(c("Standard","Deluxe","Suite"), 20, replace=TRUE)),
  satisfaction = sample(1:5, 20, replace=TRUE),
  spend_gbp    = round(rnorm(20, 180, 55), 2)
)
str(df)
'data.frame':   20 obs. of  3 variables:
 $ room_type   : Factor w/ 3 levels "Deluxe","Standard",..: 3 3 2 1 2 2 2 2 3 2 ...
 $ satisfaction: int  3 1 2 1 3 1 5 3 3 3 ...
 $ spend_gbp   : num  145 132 183 217 166 ...

factor → categorical. int → integer (could be discrete quantitative or coded ordinal). num → continuous quantitative.

Key takeaways

  • Quantitative variables measure amounts; categorical variables label groups
  • The four levels, nominal, ordinal, interval, ratio, determine valid operations
  • Discrete variables take countable values; continuous variables take any value in a range
  • The IV is what you manipulate or test; the DV is what you measure
  • Variable type determines which statistical methods are appropriate
