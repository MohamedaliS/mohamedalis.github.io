22.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Distinguish the z-test from the one-sample t-test and select appropriately.
  2. Conduct a one-sample t-test in R and interpret all output elements.
  3. State the assumptions of the one-sample t-test and assess whether they hold.
  4. Report the result of a hypothesis test in plain language with effect size.
  5. Connect the one-sample t-test to the confidence interval framework.

22.2 Introduction

Part 6 established the logic of hypothesis testing. Part 7 applies that logic to specific tests. This chapter covers the simplest case: comparing one sample mean to a known or hypothesised standard.

The dataset for this unit contains fill volume measurements for 60 bottles from a production line. The nominal fill target is 500 ml. Quality control requires that the mean fill does not differ significantly from this target.

22.3 The z-test versus the one-sample t-test

Both tests evaluate whether a sample mean differs from a hypothesised value \(\mu_0\).

The z-test applies when the population standard deviation \(\sigma\) is known:

\[z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}\]

The one-sample t-test applies when \(\sigma\) is unknown and must be estimated from the sample:

\[t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}\]

In practice, \(\sigma\) is almost never known. The one-sample t-test is therefore the standard tool. The z-test appears mainly in textbook problems and in cases where the sample is so large that the t-distribution and normal distribution are indistinguishable.

22.4 Assumptions of the one-sample t-test

  1. Independence: observations are independent of each other.
  2. Approximate normality: the population is approximately normally distributed, or the sample is large enough for the CLT to apply (typically \(n \geq 30\)).

For the fill volume data, bottles are sampled independently and \(n = 60\), so the CLT ensures that the sampling distribution of the mean is approximately normal regardless of the population shape.

22.7 Effect size: Cohen’s d for one sample

For a one-sample test, Cohen’s d measures how many standard deviations the sample mean lies from the hypothesised value:

\[d = \frac{\bar{x} - \mu_0}{s}\]

Code 
cohens_d <- (mean(df$fill_ml) - mu_0) / sd(df$fill_ml)
cat("Cohen's d =", round(cohens_d, 3), "\n")
Cohen's d = 0.044

cat("Interpretation:",
    ifelse(abs(cohens_d) < 0.2, "Negligible",
           ifelse(abs(cohens_d) < 0.5, "Small",
                  ifelse(abs(cohens_d) < 0.8, "Medium","Large"))), "\n")
Interpretation: Negligible

22.9 Connection to the confidence interval

The one-sample t-test and the 95% confidence interval always agree at \(\alpha = 0.05\): if the hypothesised value \(\mu_0\) lies outside the 95% CI, the test rejects \(H_0\); if \(\mu_0\) lies inside the CI, the test fails to reject.

Code 
cat("95% CI: [", round(ci_lo,3), ",", round(ci_hi,3), "] ml\n")
95% CI: [ 499.167 , 501.172 ] ml
Code 
cat("Hypothesised mean (mu0 =", mu_0, "):",
    ifelse(mu_0 < ci_lo | mu_0 > ci_hi,
           "OUTSIDE the CI → reject H0",
           "INSIDE the CI → fail to reject H0"), "\n")
Hypothesised mean (mu0 = 500 ): INSIDE the CI → fail to reject H0

22.10 How to report the result

A complete report of a one-sample t-test includes: the test statistic, degrees of freedom, p-value, sample mean, effect size, and confidence interval.

Mean bottle fill (\(\bar{x}\) = 500.17 ml, SD = 3.88 ml, n = 60) was compared to the nominal target of 500 ml using a one-sample t-test. The result was not statistically significant (t(59) = 0.34, p = 0.7361, d = 0.044, 95% CI [499.17, 501.17] ml).

22.11 Summary

The one-sample t-test compares a sample mean to a known or hypothesised value when the population standard deviation is unknown. The test statistic \(t = (\bar{x} - \mu_0)/(s/\sqrt{n})\) follows a t-distribution with \(n-1\) degrees of freedom. The p-value and confidence interval always agree at the same \(\alpha\). Effect size (Cohen’s d) should accompany every reported test result.

22.12 Key terms

Table 22.4: Key terms for Chapter 22
Term Definition
One-sample t-test A test comparing a sample mean to a specified value when the population SD is unknown.
Hypothesised mean (mu0) The value of the population mean specified under H0.
t statistic The standardised distance between the sample mean and mu0: (x-bar - mu0) / (s/sqrt(n)).
Degrees of freedom A parameter of the t-distribution; for a one-sample test, df = n - 1.
Cohen's d (one sample) Effect size for a one-sample test: (x-bar - mu0) / s.

## Interpreting the output

```{r}
#| label: ch22-interpretation
#| echo: false
n      <- nrow(df)
x_bar  <- mean(df$fill_ml)
s      <- sd(df$fill_ml)
t_stat <- result$statistic
p_val  <- result$p.value
ci_lo  <- result$conf.int[1]
ci_hi  <- result$conf.int[2]

interp <- data.frame(
  Element = c("t statistic", "p-value", "95% CI", "Decision (alpha=0.05)"),
  Interpretation = c(
    paste0("The sample mean (", round(x_bar,2),
           " ml) is ", abs(round(t_stat,2)),
           " standard errors below the target of ", mu_0, " ml."),
    paste0("If the true mean were 500 ml, a sample mean this far from",
           " target would occur with probability ", round(p_val,4), "."),
    paste0("We are 95% confident the true mean lies between ",
           round(ci_lo,2), " and ", round(ci_hi,2), " ml."),
    ifelse(p_val <= 0.05,
           "Reject H0. Evidence that the mean fill differs from 500 ml.",
           "Fail to reject H0. Insufficient evidence of a fill difference.")
  )
)

knitr::kable(interp,
  col.names = c("Output element", "Plain-language interpretation"),
```

## Effect size: Cohen's d for one sample

For a one-sample test, Cohen's d measures how many standard deviations the sample mean lies from the hypothesised value:

$$d = \frac{\bar{x} - \mu_0}{s}$$

```{r}
#| label: ch22-effect-size
#| echo: true
cohens_d <- (mean(df$fill_ml) - mu_0) / sd(df$fill_ml)
cat("Cohen's d =", round(cohens_d, 3), "\n")
cat("Interpretation:",
    ifelse(abs(cohens_d) < 0.2, "Negligible",
           ifelse(abs(cohens_d) < 0.5, "Small",
                  ifelse(abs(cohens_d) < 0.8, "Medium","Large"))), "\n")
```

A statistically significant result with a small effect size means the fill deficit, though detectable, may be too small to be operationally important. A result that is both significant and medium-to-large in effect size would warrant immediate action.

## Connection to the confidence interval

The one-sample t-test and the 95% confidence interval always agree at $\alpha = 0.05$: if the hypothesised value $\mu_0$ lies outside the 95% CI, the test rejects $H_0$; if $\mu_0$ lies inside the CI, the test fails to reject.

```{r}
#| label: ch22-ci-connection
#| echo: true
cat("95% CI: [", round(ci_lo,3), ",", round(ci_hi,3), "] ml\n")
cat("Hypothesised mean (mu0 =", mu_0, "):",
    ifelse(mu_0 < ci_lo | mu_0 > ci_hi,
           "OUTSIDE the CI → reject H0",
           "INSIDE the CI → fail to reject H0"), "\n")
```

## How to report the result

A complete report of a one-sample t-test includes: the test statistic, degrees of freedom, p-value, sample mean, effect size, and confidence interval.
