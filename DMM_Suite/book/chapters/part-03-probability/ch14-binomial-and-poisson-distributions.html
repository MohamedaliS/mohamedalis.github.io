  1. Identify situations that satisfy the conditions for a binomial distribution.
  2. Calculate binomial probabilities using the formula and R.
  3. Calculate the mean and standard deviation of a binomial distribution.
  4. Identify situations that satisfy the conditions for a Poisson distribution.
  5. Calculate Poisson probabilities using the formula and R.

14.2 Introduction

Chapter 13 introduced general probability distributions. Two specific distributions appear so frequently in business and management contexts that they deserve dedicated treatment. The binomial distribution models the number of successes in a fixed number of independent trials, each with the same probability of success. The Poisson distribution models the number of events occurring in a fixed interval of time or space when events occur at a constant average rate.

Both distributions are entirely determined by one or two parameters, making them practical and widely used. The dataset for this unit contains 300 accident and emergency records from a hospital, with arrival times, triage categories, wait times, and admission outcomes.

14.3 The binomial distribution

A binomial random variable X counts the number of successes in n independent trials, where each trial has probability p of success.

Four conditions must hold for the binomial model to apply:

  1. A fixed number of trials n.
  2. Each trial results in exactly one of two outcomes: success or failure.
  3. The probability of success p is the same for every trial.
  4. Trials are independent.

The probability of exactly x successes in n trials is:

\[P(X = x) = \binom{n}{x} p^x (1-p)^{n-x}\]

The mean and standard deviation are:

\[\mu = np \qquad \sigma = \sqrt{np(1-p)}\]

14.3.1 Binomial example: patient admissions

Each patient arriving at the A&E is either admitted or not. If the historical admission rate is estimated from data, the number of admissions in the next 10 arrivals follows a binomial distribution (approximately, assuming independence).

14.4 The Poisson distribution

The Poisson distribution models the number of events occurring in a fixed interval when:

  1. Events occur at a constant average rate \(\lambda\) (lambda).
  2. Events are independent of one another.
  3. Two events cannot occur at exactly the same instant.

The probability of exactly x events occurring is:

\[P(X = x) = \frac{e^{-\lambda} \lambda^x}{x!} \qquad x = 0, 1, 2, \ldots\]

The mean and variance of a Poisson random variable are both equal to \(\lambda\).

14.4.1 Poisson example: A&E arrivals per hour

Code 
# Estimate lambda from the data
# Count arrivals by hour of the day
arrivals_per_hour <- df |>
  count(hour_of_arrival, name = "count") |>
  summarise(lambda = mean(count))

lambda <- arrivals_per_hour$lambda
cat("Estimated arrival rate lambda =",
    round(lambda, 2), "patients per hour\n\n")
Estimated arrival rate lambda = 12.5 patients per hour
Code 
# Poisson probability distribution
pois_dist <- data.frame(
  x = 0:round(lambda * 3),
  probability = round(dpois(0:round(lambda * 3), lambda), 4)
)

knitr::kable(head(pois_dist, 15),

14.7 Summary

The binomial distribution applies when a fixed number of independent trials each have the same probability of success. Its mean is np and its standard deviation is \(\sqrt{np(1-p)}\). The Poisson distribution applies when events occur at a constant average rate in a fixed interval. Its mean and variance are both equal to \(\lambda\).

Both distributions are calculated in R using dbinom(), pbinom(), dpois(), and ppois(). The d prefix gives point probabilities; the p prefix gives cumulative probabilities.

14.8 Key terms

Table 14.7: Key terms for Chapter 14
Term Definition
Binomial distribution A discrete distribution counting successes in n independent trials with constant success probability p.
Success probability (p) The probability of success on a single trial in a binomial experiment.
Poisson distribution A discrete distribution counting events occurring in a fixed interval at a constant average rate.
Rate parameter (lambda) The average number of events per interval in a Poisson process.
dbinom() / dpois() R functions returning the probability that X equals exactly x.
pbinom() / ppois() R functions returning the probability that X is less than or equal to x (cumulative).

## The binomial distribution

Four conditions must hold for the binomial model to apply:

1. A fixed number of trials n.
2. Each trial results in exactly one of two outcomes: success or failure.

## The Poisson distribution

The Poisson distribution models the number of events occurring in a fixed interval when:

1. Events occur at a constant average rate $\lambda$ (lambda).
2. Events are independent of one another.
3. Two events cannot occur at exactly the same instant.

The probability of exactly x events occurring is:

$$P(X = x) = \frac{e^{-\lambda} \lambda^x}{x!} \qquad x = 0, 1, 2, \ldots$$

The mean and variance of a Poisson random variable are both equal to $\lambda$.

### Poisson example: A&E arrivals per hour

```{r}
#| label: ch14-poisson-setup
#| echo: true
# Estimate lambda from the data
# Count arrivals by hour of the day
arrivals_per_hour <- df |>
  count(hour_of_arrival, name = "count") |>
  summarise(lambda = mean(count))

lambda <- arrivals_per_hour$lambda
cat("Estimated arrival rate lambda =",
    round(lambda, 2), "patients per hour\n\n")

# Poisson probability distribution
pois_dist <- data.frame(
  x = 0:round(lambda * 3),
  probability = round(dpois(0:round(lambda * 3), lambda), 4)
)

knitr::kable(head(pois_dist, 15),
  col.names = c("Arrivals (x)", "P(X = x)"),
  caption   = paste0("Table 14.3: Poisson distribution (lambda = ",
                     round(lambda, 2), " arrivals/hour)")) |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
```

```{r}
#| label: ch14-pois-plot
#| echo: false
#| fig-height: 3.8
#| fig-cap: "Figure 14.2: Poisson distribution for hourly A&E arrivals"

  geom_bar(stat = "identity", fill = "#E8A020",
           alpha = 0.85, width = 0.6) +
  labs(x = "Arrivals per hour (x)", y = "P(X = x)",
       title = paste0("Poisson(lambda = ", round(lambda, 2), ")")) +
```
