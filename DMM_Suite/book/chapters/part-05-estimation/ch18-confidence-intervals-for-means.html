18.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Explain what a confidence interval represents.
  2. Construct a confidence interval for a population mean using the t-distribution.
  3. Interpret a confidence interval correctly and avoid common misinterpretations.
  4. Explain the effect of confidence level and sample size on interval width.
  5. Construct and compare confidence intervals across subgroups.

18.2 Introduction

Chapter 17 established that a sample mean is a point estimate of the population mean: it is a single best guess. But a single number conveys no information about how uncertain that guess is. A confidence interval replaces the point estimate with a range of plausible values, communicating both the estimate and the uncertainty surrounding it.

Confidence intervals are used constantly in applied work: in survey reports (“satisfaction is 3.8, margin of error ±0.2”), in clinical trials, in market research, and in operations analysis. The dataset for this unit contains 240 food delivery records, with variables including delivery time, distance, cuisine type, and driver rating.

18.3 What a confidence interval is

A confidence interval is a range of values constructed from sample data such that, if the sampling procedure were repeated many times, a specified proportion of the intervals would contain the true population parameter.

A 95% confidence interval means: if we were to draw many independent samples and construct a confidence interval from each, approximately 95% of those intervals would contain the true population mean. It does not mean there is a 95% probability that the true mean lies in this specific interval - once the interval is computed, the true mean either is or is not inside it.

This distinction matters. The confidence refers to the long-run reliability of the procedure, not to the probability of a single outcome.

18.4 Constructing a confidence interval for the mean

When the population standard deviation is unknown (which is almost always the case in practice), the confidence interval for the population mean uses the t-distribution:

\[\bar{x} \pm t^* \cdot \frac{s}{\sqrt{n}}\]

where \(t^*\) is the critical value from the t-distribution with \(n-1\) degrees of freedom, and \(s/\sqrt{n}\) is the estimated standard error.

Code 
library(dplyr)

n      <- nrow(df)
x_bar  <- mean(df$delivery_minutes)
s      <- sd(df$delivery_minutes)
se     <- s / sqrt(n)
t_star <- qt(0.975, df = n - 1)   # 95% CI, two-tailed

lower_95 <- x_bar - t_star * se
upper_95 <- x_bar + t_star * se

ci_table <- data.frame(
  Statistic = c("Sample size (n)", "Sample mean", "Sample SD",
                "Standard error", "t* (df = n-1)", "Lower bound (95%)",
                "Upper bound (95%)"),
  Value = round(c(n, x_bar, s, se, t_star, lower_95, upper_95), 3)
)

knitr::kable(ci_table,
  col.names = c("Statistic", "Value"),
  caption   = "Table 18.1: 95% confidence interval for mean delivery time") |>
  kableExtra::kable_styling(bootstrap_options = c("striped"),
                             full_width = FALSE)
Table 18.1: 95% confidence interval for mean delivery time
Statistic Value
Sample size (n) 240.000
Sample mean 36.281
Sample SD 8.666
Standard error 0.559
t* (df = n-1) 1.970
Lower bound (95%) 35.179
Upper bound (95%) 37.383

Interpretation. Based on these 240 deliveries, we estimate the population mean delivery time to be between 35.2 and 37.4 minutes. We use a procedure that produces intervals containing the true mean 95% of the time in the long run.

18.6 Effect of confidence level on interval width

A higher confidence level requires a wider interval to capture the true parameter more reliably.

Code 
levels_df <- do.call(rbind, lapply(c(0.80, 0.90, 0.95, 0.99), function(cl) {
  res <- t.test(df$delivery_minutes, conf.level = cl)
  data.frame(
    Confidence_level = paste0(cl * 100, "%"),
    Lower = round(res$conf.int[1], 2),
    Upper = round(res$conf.int[2], 2),
    Width = round(diff(res$conf.int), 2)
  )
}))

knitr::kable(levels_df,
  col.names = c("Confidence level", "Lower bound",
                "Upper bound", "Interval width (mins)"),
  caption   = "Table 18.3: Confidence intervals at different confidence levels") |>
  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 18.3: Confidence intervals at different confidence levels
Confidence level Lower bound Upper bound Interval width (mins)
80% 35.56 37.00 1.44
90% 35.36 37.20 1.85
95% 35.18 37.38 2.20
99% 34.83 37.73 2.91

Increasing confidence from 95% to 99% widens the interval. There is a trade-off: higher confidence comes at the cost of less precision.

18.7 Effect of sample size on interval width

For a fixed confidence level, a larger sample produces a narrower interval because the standard error decreases.

18.8 Confidence intervals for subgroups

Code 
ci_by_weekend <- df |>
  group_by(weekend) |>
  summarise(
    n      = n(),
    mean   = round(mean(delivery_minutes), 2),
    lower  = round(t.test(delivery_minutes)$conf.int[1], 2),
    upper  = round(t.test(delivery_minutes)$conf.int[2], 2)
  )

                "Lower 95% CI", "Upper 95% CI"),

  kableExtra::kable_styling(bootstrap_options = c("striped", "hover"),
                             full_width = FALSE)
Table 18.4: Confidence intervals for weekday vs weekend deliveries
Weekend n Mean (mins) Lower 95% CI Upper 95% CI
No 146 34.64 33.29 35.99
Yes 94 38.83 37.04 40.62

If the two confidence intervals do not overlap, there is good informal evidence that the population means differ. Overlapping intervals are less conclusive: they do not rule out a difference. A formal two-sample test, covered in Chapter 23, provides a definitive answer.

18.9 Common misinterpretations

Table 18.5: Common misinterpretations of confidence intervals
Incorrect interpretation Correct interpretation
There is a 95% probability that the true mean lies in this interval. The interval was constructed using a procedure that captures the true mean in 95% of repeated applications. For this specific interval, the true mean either is or is not inside it.
95% of individual delivery times fall within the interval. The interval estimates the population mean, not the spread of individual observations. Individual times vary far more widely.
A wider interval is always less useful than a narrower one. A wider interval at a higher confidence level may be more honest about the true uncertainty than a narrower interval at a lower level.

18.10 Summary

A confidence interval provides a range of plausible values for a population parameter, communicating both the estimate and its uncertainty. For a population mean with unknown \(\sigma\), the interval is \(\bar{x} \pm t^* \cdot (s/\sqrt{n})\). Confidence refers to the long-run reliability of the procedure, not to the probability that the true mean lies in any specific computed interval.

A higher confidence level produces a wider interval. A larger sample produces a narrower interval for a fixed confidence level. These trade-offs inform sample size planning in survey and experimental design.

18.11 Key terms

Table 18.6: Key terms for Chapter 18
Term Definition
Confidence interval A range of values constructed from sample data using a procedure that captures the true parameter a specified percentage of the time.
Confidence level The proportion of intervals that would contain the true parameter if the sampling procedure were repeated many times; typically 90%, 95%, or 99%.
t-distribution A family of symmetric bell-shaped distributions indexed by degrees of freedom; used when sigma is unknown.
Critical value (t*) The t-value such that the area between -t* and t* under the t-distribution equals the confidence level.
Margin of error Half the width of the confidence interval: t* times the standard error.
Degrees of freedom A parameter of the t-distribution equal to n-1 for a one-sample mean interval.

18.12 Exercises

  1. A sample of 36 call durations has mean 4.8 minutes and standard deviation 1.6 minutes. Construct a 95% confidence interval for the population mean. Interpret the interval in plain language.

  2. Using Table 18.3, what confidence level would you recommend for a food delivery company reporting average delivery times to customers? Justify your answer.

  3. Using Table 18.4, the confidence intervals for weekday and weekend deliveries overlap. Does this mean weekend and weekday mean delivery times are the same? What would you do next?

  4. A researcher doubles her sample size from 60 to 120. By approximately what factor does the width of her 95% confidence interval change?

  5. A report states: “We are 95% confident that the true mean is in the interval [28.4, 31.6] minutes.” A colleague says: “That means 95% of deliveries take between 28.4 and 31.6 minutes.” How would you correct this?

## Effect of sample size on interval width

For a fixed confidence level, a larger sample produces a narrower interval because the standard error decreases.

```{r}
#| label: ch18-ci-samplesize
#| echo: false
#| fig-height: 3.5
#| fig-cap: "Figure 18.1: Interval width narrows as sample size increases"
set.seed(99)
width_df <- do.call(rbind, lapply(c(20, 40, 60, 80, 120, 200), function(n_s) {
  samp <- sample(df$delivery_minutes, n_s)
  res  <- t.test(samp, conf.level = 0.95)
  data.frame(n = n_s, width = round(diff(res$conf.int), 2))
}))

ggplot(width_df, aes(x = n, y = width)) +
  geom_line(colour = "#1B3A5C", linewidth = 1.1) +
  geom_point(colour = "#C0392B", size = 3) +
  labs(x = "Sample size (n)", y = "95% CI width (minutes)",
       title = "Diminishing returns: width narrows with larger samples") +
  theme_dmm()
```
