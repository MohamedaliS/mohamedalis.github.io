21.1 Learning objectives

By the end of this chapter, the student should be able to:

  1. Distinguish between one-tailed and two-tailed tests and select appropriately.
  2. Interpret a p-value correctly and avoid common misinterpretations.
  3. Explain the relationship between alpha, the p-value, and the rejection decision.
  4. Calculate and interpret a simple effect size measure.
  5. Distinguish statistical significance from practical significance.

21.2 Introduction

Chapter 20 introduced the framework: null hypothesis, alternative hypothesis, and error types. This chapter fills in the remaining concepts needed to interpret any hypothesis test: the direction of the test, the p-value that quantifies evidence, the effect size that measures magnitude, and the distinction between a statistically significant result and a practically important one.

The dataset for this unit contains results from an A/B test of two versions of a product checkout page. 300 users were randomly assigned to version A (current) or version B (redesigned). Variables include session duration, pages viewed, conversion rate, and order value.

21.3 One-tailed versus two-tailed tests

The alternative hypothesis determines whether a test is one-tailed or two-tailed.

A two-tailed test is used when the research question is whether two groups differ, without specifying a direction: \[H_1: \mu_1 \neq \mu_2\]

A one-tailed test is used when the research question specifies a direction: \[H_1: \mu_1 > \mu_2 \quad \text{or} \quad H_1: \mu_1 < \mu_2\]

Table 21.1: One-tailed versus two-tailed test selection
Test type H1 Use when Note
Two-tailed mu1 ≠ mu2 Direction of difference is unknown or unspecified. Default for most business tests; more conservative.
One-tailed (upper) mu1 > mu2 Specifically hypothesising that group 1 is higher. Should be specified before seeing data; not chosen post-hoc.
One-tailed (lower) mu1 < mu2 Specifically hypothesising that group 1 is lower. As above.

For the A/B test in this unit, the question is whether version B has a different conversion rate from version A - direction unspecified - so a two-tailed test is appropriate.

21.4 The p-value

The p-value is the probability of observing a test statistic as extreme as the one computed, or more extreme, assuming \(H_0\) is true.

It is not the probability that \(H_0\) is true. It is not the probability of having made an error. It is a measure of how surprising the observed data are under the null hypothesis.

A small p-value means the observed result would be rare if \(H_0\) were true. A large p-value means the observed result is consistent with \(H_0\).

21.4.1 The decision rule

If \(p \leq \alpha\), reject \(H_0\). If \(p > \alpha\), fail to reject \(H_0\).

21.5 Common p-value misinterpretations

Table 21.4: Common p-value misinterpretations
Incorrect statement Correct interpretation
The p-value is the probability that H0 is true. The p-value is the probability of observing a result this extreme if H0 were true. It says nothing about the probability that H0 is true.
A p-value of 0.04 means there is a 4% chance we are wrong. The p-value is not an error probability for a single test. It is a long-run frequency statement.
A p-value > 0.05 proves H0 is true. Failing to reject H0 means the data are consistent with H0, not that H0 is proved. Absence of evidence is not evidence of absence.
A p-value < 0.05 means the finding is important. Statistical significance depends on sample size. A trivial effect can be statistically significant in a large sample. Practical importance requires separate assessment.

21.6 Effect size

Statistical significance tells you whether an effect is detectable. Effect size tells you how large it is. A result can be statistically significant without being practically meaningful, particularly in large samples.

For the difference between two proportions, the simplest effect size is the absolute difference:

\[\text{Effect size} = \hat{p}_B - \hat{p}_A\]

For differences between means, Cohen’s d is commonly used:

\[d = \frac{\bar{x}_1 - \bar{x}_2}{s_{\text{pooled}}}\]

Code 
p_A   <- x_A / n_A
p_B   <- x_B / n_B
abs_diff <- p_B - p_A
rel_diff <- abs_diff / p_A

# Cohen's d for session minutes
m_A  <- mean(df$session_minutes[df$version == "A"])
m_B  <- mean(df$session_minutes[df$version == "B"])
s_p  <- sqrt(((n_A-1)*var(df$session_minutes[df$version=="A"]) +
               (n_B-1)*var(df$session_minutes[df$version=="B"])) /
              (n_A + n_B - 2))
cohens_d <- (m_B - m_A) / s_p

es_table <- data.frame(
  Measure     = c("Absolute difference in conversion (B - A)",
                  "Relative difference in conversion",
                  "Cohen's d for session minutes"),
  Value       = round(c(abs_diff, rel_diff, cohens_d), 3),
  Interpretation = c(
    paste0(round(abs_diff * 100, 1), " percentage points"),
    paste0(round(rel_diff * 100, 1), "% relative increase"),
    ifelse(abs(cohens_d) < 0.2, "Negligible",
           ifelse(abs(cohens_d) < 0.5, "Small",
                  ifelse(abs(cohens_d) < 0.8, "Medium", "Large")))
  )
)

21.7 Statistical vs practical significance

Table 21.7: Statistical vs practical significance
Scenario p-value verdict Practical importance Recommended action
Large sample, tiny effect < 0.05 (significant) Not meaningful Report the effect size; question practical value.
Small sample, large effect > 0.05 (not significant) Potentially important but hard to detect Increase sample size; do not discard the finding.
Large sample, large effect < 0.05 (significant) Both significant and meaningful Strong evidence of an important effect.
Small sample, tiny effect > 0.05 (not significant) Neither significant nor meaningful No basis for action from this test.

21.8 Power

The power of a test is the probability of correctly rejecting a false \(H_0\). Power is determined by three factors: sample size, effect size, and significance level \(\alpha\).

\[\text{Power} = 1 - \beta\]

A study with low power has a high risk of missing a real effect. Power analysis before data collection determines the sample size needed to detect a specified effect size with a specified probability.

Code 
# Power for the conversion rate test
# Using pwr package if available, otherwise illustrative calculation
library(dplyr)
cat("Observed effect size (abs diff):", round(abs_diff, 3), "\n")
Observed effect size (abs diff): 0.113
Code 
cat("Sample size per group:", n_A, "\n")
Sample size per group: 150
Code 
# Approximate power using normal approximation
p_pool <- (x_A + x_B) / (n_A + n_B)
se_h0  <- sqrt(p_pool * (1 - p_pool) * (1/n_A + 1/n_B))
z_obs  <- abs_diff / se_h0
z_crit <- qnorm(0.975)
power_approx <- pnorm(z_obs - z_crit) + pnorm(-z_obs - z_crit)

cat("Approximate power:", round(power_approx, 3), "\n")
Approximate power: 0.593
Code 
cat("Interpretation: ~", round(power_approx * 100), "% chance of detecting",
    "\nthis effect size with n =", n_A, "per group at alpha = 0.05\n")
Interpretation: ~ 59 % chance of detecting 
this effect size with n = 150 per group at alpha = 0.05

21.9 Summary

One-tailed tests specify a direction in \(H_1\); two-tailed tests do not. The p-value measures how surprising the data are under \(H_0\): a small p-value indicates the data would be unusual if \(H_0\) were true. It is not the probability that \(H_0\) is true.

Effect size quantifies the magnitude of an observed difference independently of sample size. Statistical significance (p < \(\alpha\)) indicates that an effect is detectable; practical significance requires a separate assessment of whether the effect is large enough to matter. Power is the probability of detecting a real effect; it is determined by sample size, effect size, and \(\alpha\).

21.10 Key terms

Table 21.8: Key terms for Chapter 21
Term Definition
Two-tailed test A hypothesis test where H1 specifies a difference in either direction.
One-tailed test A hypothesis test where H1 specifies the direction of the expected difference.
p-value The probability of observing a result as extreme as the one obtained, or more extreme, assuming H0 is true.
Effect size A standardised measure of the magnitude of an observed difference, independent of sample size.
Cohen's d A standardised effect size for the difference between two means: (mean1 - mean2) / pooled SD.
Statistical significance A result is statistically significant when p <= alpha; the evidence is sufficient to reject H0.
Practical significance A result is practically significant when the effect is large enough to matter for real decisions.
Power The probability of correctly rejecting a false H0; equals 1 - beta.

## Power

The **power** of a test is the probability of correctly rejecting a false $H_0$. Power is determined by three factors: sample size, effect size, and significance level $\alpha$.

$$\text{Power} = 1 - \beta$$

A study with low power has a high risk of missing a real effect. Power analysis before data collection determines the sample size needed to detect a specified effect size with a specified probability.

```{r}
#| label: ch21-power
#| echo: true
# Power for the conversion rate test
# Using pwr package if available, otherwise illustrative calculation
library(dplyr)
cat("Observed effect size (abs diff):", round(abs_diff, 3), "\n")
cat("Sample size per group:", n_A, "\n")

# Approximate power using normal approximation
p_pool <- (x_A + x_B) / (n_A + n_B)
se_h0  <- sqrt(p_pool * (1 - p_pool) * (1/n_A + 1/n_B))
z_obs  <- abs_diff / se_h0
z_crit <- qnorm(0.975)
power_approx <- pnorm(z_obs - z_crit) + pnorm(-z_obs - z_crit)

cat("Approximate power:", round(power_approx, 3), "\n")
cat("Interpretation: ~", round(power_approx * 100), "% chance of detecting",
    "\nthis effect size with n =", n_A, "per group at alpha = 0.05\n")
```

## Summary

One-tailed tests specify a direction in $H_1$; two-tailed tests do not. The p-value measures how surprising the data are under $H_0$: a small p-value indicates the data would be unusual if $H_0$ were true. It is not the probability that $H_0$ is true.

Effect size quantifies the magnitude of an observed difference independently of sample size. Statistical significance (p < $\alpha$) indicates that an effect is detectable; practical significance requires a separate assessment of whether the effect is large enough to matter. Power is the probability of detecting a real effect; it is determined by sample size, effect size, and $\alpha$.
