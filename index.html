Web Tools

Small browser-based utilities built for practical teaching and everyday decision support. The current collection includes Price Memory for local grocery price tracking and Margin Guard for promotion break-even analysis.

Browse tool source on GitHub →

Price Memory

Live

A single-file grocery price tracker that logs item, store, price, and date locally in the browser. It uses IndexedDB for storage and a lightweight canvas chart to show whether a deal is actually below your usual price.

PWAIndexedDBCanvas APIVanilla JS
GitHub Pages App Open App →

Margin Guard

Live

A promotion break-even calculator for pricing decisions. It compares baseline contribution, discounted unit margin, and expected volume lift so you can see whether a campaign actually earns back the margin it gives away.
