Web Tools

Standalone browser-based tools built for quick, useful decisions. Price Memory turns your own purchase log into a grocery price baseline, while Margin Guard checks whether a promotion can recover the contribution margin it gives away.

Browse web tool repositories on GitHub →

Price Memory

Live

A local grocery price tracker built as a single-page web app. It stores item, store, price, and date in IndexedDB so you can build a personal price history and check whether a sale is actually below your usual price.

Margin Guard

Live

A single-page promotion break-even calculator for pricing and campaign decisions. It compares baseline contribution against promo-period contribution and shows discount depth, promo contribution per unit, break-even units, required unit lift, and expected net contribution delta.
