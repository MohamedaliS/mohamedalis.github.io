Web Tools

Standalone browser-based tools built for quick, useful decisions. Price Memory turns your own purchase log into a grocery price baseline, while Margin Guard checks whether a promotion can recover the contribution margin it gives away.

Browse web tool repositories on GitHub →

Price Memory

Live

A single-file progressive web app for tracking grocery prices over time. Runs entirely in the browser, with no backend and no account. Features IndexedDB storage, a Canvas price chart, and a 16-currency onboarding selector. Deploys directly to GitHub Pages.

A browser-based pricing and promotion calculator that shows discount depth, promo contribution per unit, break-even units, required lift, and expected net contribution delta. Scenarios are saved locally in the browser for quick comparison.
